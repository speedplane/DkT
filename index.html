A New Way to Browse PACER

DkT is a feature-rich, custom PACER/ECF client for iOS.

Efficiently search dockets, bookmark and update dockets, view and save documents, manage logins, batch downloading, and more.

Bookmarking a docket will save it to the device, allow offline access, and accessing the saved without cost. Bookmarked dockets can be updated for a minimsl PACER fee.

DkT is a custom iOS client for PACER. The application directly interacts with PACER while adding a custom user interface. DkT works with the District, Bankruptcy, and Appellate courts. DkT does not communicate data to any third-party or server, other than PACER.

DkT is free, but PACER charges still apply for viewing unsaved dockets and documents. If you enjoy using DkT and would like to donate, please consider a small donation to Gideon's Promise or Chai Lifeline. DkT is open-source and includes the work of many others, as described on the project page.
