Introducing DkT

DkT is an open-source client for PACER/ECF for iOS. DkT overlays PACER with a custom user interface, mediates network requests with the PACER/ECF, and parses results to facilitate access to federal dockets on iOS devices. DkT works with the District, Bankruptcy, and Appellate courts, and also adds a number of convenient features to PACER such as bookmarking, saving, batch downloading, and more.
